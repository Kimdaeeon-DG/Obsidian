### 유저를 그룹에 넣는 방법
1. 사용자가 존재하는지 확인:
```bash
getent passwd inam
```
2. 사용자가 존재하지 않으면 새 사용자 생성 :
```bash
sudo adduser inam
```
3. 사용자를 그룹에 추가:
```bash
sudo usermod -aG user42 inam
```
